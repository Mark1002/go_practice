# go practice
This is my golang practice project.

1. create go demo code package
```
$ mkdir <package>
$ cd <package>
```
2. import go package at outter main.go file
```go
package main

import (
	"github.com/mark1002/go_practice/<package>"
)

func main() {
	<package>.Excute()
}
```
3. run all practice
```
<module> $ go run main.go
```
